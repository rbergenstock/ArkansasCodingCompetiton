Competition Strategy

⏱ Time Management (3 Hours)

180 minutes. Spend 5 min scanning ALL challenges. Sort by difficulty. Assign by strength. Easy first.

Past competitions: 7 challenges from 4 to 28 pts. Get easy wins in 20 min, then divide and conquer.

🖥 One-Computer Workflow

Paper-first method (proven winners): Each member works a separate problem on paper. Write actual code. When your turn at keyboard, you're just typing. Rotate efficiently.

📋 Format Matters

Common traps: Missing colon/space. Case sensitivity. Not handling missing files or invalid input. Read the rubric — it shows partial credit.

🎯 Points

4 pts — Print team info (free) · 8-14 — String/encoding/validation · 16-22 — File I/O, JSON/CSV · 22-28 — Complex ciphers

Partial credit is real! Getting prompt/input handling (2 pts) is always worth it.

🧠 Key Topics

Strings: substring, replace, case, ASCII · File I/O: JSON, CSV, existence checks, append · Encoding: Base conversion, Caesar variants · MC: Design patterns, Big-O, CIA triad, SQL, OOP, bitwise, base conversion
